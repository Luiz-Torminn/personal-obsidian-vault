# How to use the Inbox

> [!INFO] This is the capture zone
> Drop quick, raw thoughts here — one note per idea. Fleeting notes are *temporary*: they exist to be processed, not kept.

## The flow

1. **Capture** — make a new note in `+ Inbox/` the moment an idea, question, or quote appears. Don't polish it.
2. **Process** (regularly) — for each fleeting note, decide:
   - **Promote** → rewrite it as a `type: permanent` note in `Permanent/`, give it `id` + `up:` + links, file it under the right MOC, then **delete the fleeting note**.
   - **Defer** → if it's a topic to learn later, log it in [[New Studies]] and delete.
   - **Discard** → if it no longer matters, just delete.
3. **Empty the Inbox** — a healthy Inbox trends toward zero.

> [!TIP]
> Keep this note here as a reminder. Everything else in `+ Inbox/` should be transient.
